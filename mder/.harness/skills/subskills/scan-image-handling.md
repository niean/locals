# Subskill: 扫描图片处理

## 任务

扫描源代码检查图片处理约束，逐文件读取，输出违规清单。

## 输入

- {files}：文件路径列表（可选）。未提供时扫描默认范围。

## 默认扫描范围

- index.html

## 检查规则

当前项目不涉及图片资源管理，暂无检查规则。

## 已知例外（不视为违规）

（暂无）

## 输出格式

```
## 图片处理 扫描结果
违规数量：N
| # | 文件 | 行号 | 违规项 | 建议修复 |
|---|------|------|--------|---------|
未发现违规的检查项：...
```
