## 知识库设计哲学

knowledge/ 按"知识用途"而非"功能领域"组织，编号分段体现两类文件的不同性质：

- 01~05 认知与约束类：为 AI 建立项目认知（概览、架构、约定、数据、模式），按关注点拆分而非集中为单一 design.md，使 AI 按需加载时能精确命中所需片段
- 21~22 工具与索引类：为 AI 提供快速查找能力（术语字典、文件导航），避免逐文件搜索
