# 产品需求 - 稳定固化(不频繁变更)

## 1. 极简摘要

浏览器端Markdown文件渲染工具，支持打开本地.md/.markdown/.txt文件并以GitHub风格渲染展示，提供代码语法高亮和文件元信息显示。

产品定位、体验原则与判断准则见 prd/01-prd-sense.md。

---

## 2. 页面结构

单页面应用，从上到下：
1. Header：标题"MD文件渲染工具" + "重新加载"按钮（文件已打开时显示）+ "打开文件"按钮
2. 渲染区：Markdown渲染结果展示区，默认显示空状态引导文字
3. 文件信息栏：底部右对齐显示文件路径、行数、大小、修改时间

---

## 3. 功能需求

### 3.1 文件打开
- 支持File System Access API（现代浏览器）打开文件
- 不支持时自动降级为传统file input
- 支持文件类型：.md、.markdown、.txt
- 保存文件句柄用于重新加载

### 3.2 Markdown渲染
- 使用marked.js解析Markdown，启用GFM和breaks
- 使用highlight.js对代码块进行语法高亮（GitHub风格）
- 渲染结果包含：标题、段落、列表、表格、代码块、引用、链接、图片、分隔线
- 渲染完成后显示文件元信息（路径、行数、字节数、最后修改时间）
- 支持"重新加载"按钮刷新当前文件内容

---

## 4. 非功能与约束

- 纯前端实现，无后端依赖
- 单HTML文件 + lib/目录第三方库
- 响应式适配（移动端 max-width: 768px）
- 浏览器兼容：Chrome、Safari、Firefox、Edge最新版
- 文件编码：UTF-8
