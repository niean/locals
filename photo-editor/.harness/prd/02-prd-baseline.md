# 产品基线

## 约束
本文仅供自然人使用，未经人工确认、禁止AI修改。

## 功能清单

- 裁剪：自由裁剪、原始比例、常用比例（1:1, 4:3, 3:4, 16:9, 9:16）
- 画笔：自由绘制，可调大小和颜色
- 文字标注：点击添加文字，可调字号、颜色、字体
- 马赛克：拖拽区域马赛克，可调块大小
- 形状：箭头、矩形框标注，可调颜色和线宽
- 撤销/重做：Ctrl+Z / Ctrl+Y，最大 50 步
- 图片加载：按钮打开 或 拖拽图片到画布
- 图片导出：下载编辑后图片，格式可选（PNG/JPG/WebP），默认与输入一致

## 界面布局

应用采用三栏布局结构：

- 左侧工具栏：裁剪、画笔、文字、马赛克、形状五种编辑工具切换按钮
- 顶部属性栏：工具参数选项（根据当前工具动态显示）、逆时针旋转按钮、撤销/重做按钮、打开图片按钮、导出格式选择、下载按钮
- 中间画布区域：图片编辑主区域，包含画布和占位提示

## 产品约束

- 纯客户端应用，无后端服务
- 零外部依赖，无构建系统
- 支持输入格式：JPG、PNG、WebP
- 最大图片尺寸：4096x4096
- 技术规范：JS Airbnb Style Guide、CSS ITCSS 架构 + BEM 命名
- 推荐浏览器：Chrome、Firefox、Safari、Edge 最新版
