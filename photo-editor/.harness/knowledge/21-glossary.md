# 术语表

| 术语 | 含义 |
|------|------|
| ITCSS | Inverted Triangle CSS，CSS 架构分层方法（Settings > Tools > Generic > Elements > Objects > Components > Utilities） |
| BEM | Block-Element-Modifier，CSS 命名约定（.block__element--modifier） |
| overlay canvas | 叠在主画布上方的透明画布，用于临时视觉反馈（裁剪选区等） |
| ImageData | Canvas API 提供的像素数据对象，用于快照存储和恢复 |
| BaseTool | 工具基类，定义工具生命周期（activate/deactivate/mouse events） |
